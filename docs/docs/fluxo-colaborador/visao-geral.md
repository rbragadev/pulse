# 🧭 Fluxo do Colaborador

Este guia explica, passo a passo, como um colaborador OTG usa a plataforma Pulse — desde o primeiro acesso até o uso avançado.

---

## Jornada completa

```
1. Primeiro acesso → Login com Google corporativo
2. Explorar o feed → Ver reconhecimentos da empresa
3. Enviar um kudos → Reconhecer um colega
4. Interagir → Reagir e comentar no feed
5. Explorar perfis → Ver reputação de colegas
6. Acompanhar ranking → Galácticos do mês
7. Colecionar badges → Completar conquistas
8. Construir reputação → Ser reconhecido com frequência
```

---

## Resumo de todas as ações disponíveis

| Ação | Onde fazer | Resultado |
|------|-----------|---------|
| Login | Tela de Login | Acesso autenticado |
| Enviar kudos | Feed (composer) ou botão "Reconhecer" | Publicação no feed + pontos para o destinatário |
| Curtir kudos | Feed — botão ❤️ | Like registrado |
| Reagir | Feed — 5 reações | Reação exibida no card |
| Comentar | Feed — campo de comentário | Comentário público no card |
| Ver perfil | Galácticos / clicar no nome | Perfil completo do colaborador |
| Votar na reputação | Perfil de qualquer colega | +1 na dimensão votada |
| Ver ranking | Galácticos | Top mensal e geral |
| Ver badges | Menu "Badges" | Badges conquistadas e bloqueadas |
| Ver conquistas | Menu "Badges" | Timeline de conquistas |
| Acompanhar feed | Menu "Feed" | Todos os reconhecimentos em tempo real |

---

## Boas práticas para colaboradores

:::tip Como escrever um bom Kudos
- **Seja específico**: mencione a situação concreta
- **Inclua o impacto**: o que aquele comportamento gerou para o time ou cliente
- **Seja autêntico**: escreva como você realmente sente
- **Reconheça pessoas diferentes**: não reconheça sempre as mesmas pessoas

> ❌ "Parabéns pelo trabalho!"
>
> ✅ "O Lucas resolveu um bug crítico em produção em menos de 2 horas, salvando o lançamento do produto. Trabalho em equipe e calma sob pressão são raros — muito obrigado!"
:::

## Frequência recomendada

| Comportamento | Frequência ideal |
|--------------|----------------|
| Enviar kudos | 2-3 por semana |
| Reagir no feed | Diariamente |
| Comentar | Sempre que se sentir movido |
| Visitar perfis | Semanalmente |
| Checar ranking | Semanalmente |
