# 🚀 Roadmap — Próximas Evoluções do Pulse

Esta seção documenta o que está planejado para as próximas versões do Pulse. O roadmap é vivo e pode ser atualizado com base no feedback dos colaboradores e da liderança.

---

## V1 — Atual ✅

O Pulse V1 está **em produção** com as seguintes funcionalidades:

- [x] Autenticação Google com domínio restrito @grupootg.com
- [x] Feed social de kudos com scroll infinito
- [x] Criação de kudos com categoria e mensagem
- [x] Reações emocionais (🔥🚀❤️👏🧠)
- [x] Comentários no feed
- [x] Likes
- [x] Ranking de Galácticos (mensal + geral)
- [x] Pódio olímpico (2-1-3)
- [x] Hall da Fama com histórico de temporadas
- [x] Sistema de Temporadas
- [x] Perfil social completo (Orkut-style)
- [x] Reputação social (3 dimensões votáveis)
- [x] 8 Badges com raridades (Comum → Lendário)
- [x] 8 Conquistas desbloqueáveis
- [x] Widgets no feed (ranking, comunidades, aniversários, eventos)
- [x] Notificações visuais (bell) — preview
- [x] Painel Admin completo
- [x] Analytics de engajamento
- [x] Moderação de conteúdo

## V2 — Próxima versão 🔜

**Foco:** Engajamento em tempo real e comunidades

| Feature | Prioridade | Descrição |
|---------|-----------|-----------|
| 🔔 Notificações em tempo real | Alta | WebSocket ou SSE — notificar ao receber kudos, comentário ou reação |
| 👥 Comunidades funcionais | Alta | Grupos temáticos com feed próprio e membros |
| 🎂 Aniversários automáticos | Média | Buscar data de aniversário do colaborador e gerar lembretes reais |
| 📅 Eventos corporativos | Média | Integração com Google Calendar para eventos da OTG |
| 📱 Notificações por e-mail | Média | Digest semanal com os principais reconhecimentos |
| 🏅 Badges automáticas | Alta | Atribuição automática via triggers (sem precisar de admin) |
| 🔍 Busca global | Média | Buscar por colaboradores, kudos, badges |
| 📲 PWA | Baixa | Progressive Web App para uso em mobile |

## V3 — Integrações externas 🌐

**Foco:** Expandir o Pulse para onde os colaboradores já estão

| Feature | Prioridade | Descrição |
|---------|-----------|-----------|
| 💬 Integração Slack | Alta | `/kudos @pessoa mensagem` diretamente no Slack |
| 🟦 Integração Microsoft Teams | Alta | Bot de reconhecimento no Teams |
| 📊 Export de dados | Média | CSV/Excel de kudos para People e liderança |
| 🔗 SSO corporativo | Média | Suporte a Azure AD / Okta além do Google |
| 🗓️ Integração Google Calendar | Baixa | Eventos e aniversários automáticos |

## V4 — IA e Gamificação Avançada 🤖

**Foco:** Plataforma inteligente que aprende e sugere

| Feature | Prioridade | Descrição |
|---------|-----------|-----------|
| 🤖 Sugestões de kudos por IA | Alta | "Você trabalhou com Ana esta semana — que tal reconhecê-la?" |
| 📝 Melhoria de texto com IA | Média | Ajuda a escrever mensagens mais impactantes |
| 🏆 Marketplace de recompensas | Alta | Colaboradores trocam pontos por benefícios reais |
| 🎮 Missões e desafios semanais | Média | Gamificação avançada com missões temporárias |
| 📈 Análise de sentimento | Baixa | Detectar tendências de cultura via NLP |
| 🌡️ Health Score de cultura | Média | Score geral da saúde cultural por departamento |
| 🎯 Metas de reconhecimento | Média | Definir metas de reconhecimento por time |
