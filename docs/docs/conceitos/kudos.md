# 🚀 Kudos

**Kudos** são o coração do Pulse. São mensagens de reconhecimento que um colaborador envia para outro, destacando uma contribuição específica, um comportamento exemplar ou um resultado extraordinário.

---

## Anatomia de um Kudos

Um kudos é composto por:

| Campo | Descrição | Obrigatório |
|-------|-----------|-------------|
| **Autor** | Quem enviou o reconhecimento | Sim (automático) |
| **Destinatário** | Quem recebeu o reconhecimento | Sim |
| **Mensagem** | Texto descritivo do reconhecimento | Sim (mín. 10 chars) |
| **Categoria** | Eixo de valor da OTG que foi demonstrado | Sim |
| **Data** | Quando foi publicado | Sim (automático) |

---

## Categorias disponíveis

Cada kudos precisa ter uma categoria que representa o valor da OTG demonstrado:

| Categoria | Ícone | Peso | Descrição |
|-----------|-------|------|-----------|
| Trabalho em Equipe | 🤝 | 1.0 | Colaboração, ajuda mútua, sinergia |
| Inovação | 💡 | 1.5 | Novas ideias, soluções criativas |
| Liderança | 👑 | 1.5 | Influência positiva, gestão de pessoas |
| Resultado | 📊 | 1.2 | Entregas concretas, métricas, impacto |
| Cliente | 🎯 | 1.3 | Foco no cliente, satisfação, suporte |
| Cultura | 🌟 | 1.0 | Valores OTG, diversidade, bem-estar |

:::tip Peso das categorias
O peso multiplica os pontos recebidos. Um kudos de "Inovação" (peso 1.5) gera mais pontos no ranking do que um de "Trabalho em Equipe" (peso 1.0).
:::

---

## O que acontece quando um Kudos é enviado?

```
Colaborador A envia kudos para Colaborador B
          ↓
1. Kudos publicado no feed (visível para todos)
2. Destinatário recebe pontos no ranking (baseado na categoria)
3. Likes, reações e comentários ficam disponíveis
4. Sistema verifica conquistas desbloqueáveis
5. Temporada ativa computa os pontos
```

---

## Como interagir com um Kudos?

Além de visualizar, qualquer colaborador pode:

- ❤️ **Curtir** — like simples no reconhecimento
- 🔥🚀❤️👏🧠 **Reagir** — 5 tipos de reações emocionais
- 💬 **Comentar** — mensagem pública visível no card
- 🔗 **Compartilhar** — (em roadmap)

---

## Restrições

:::warning Regras anti-spam
- Não é possível enviar kudos para si mesmo
- Múltiplos kudos para a mesma pessoa no mesmo dia são permitidos, mas monitorados pelo People
- O conteúdo pode ser moderado por admins
:::
